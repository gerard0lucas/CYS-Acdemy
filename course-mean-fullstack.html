Course Overview

Enterprise full stack with Angular (frontend), Node.js, Express (backend), and MongoDB. TypeScript, reactive forms, HTTP client, guards, interceptors. Build enterprise apps/CRM with testing, DevOps overview, and deployment. Portfolio and interview prep.

Introduction to Web Development, Client vs Server, How the Internet Works, JavaScript: Variables, Data Types, Operators, Control Statements, Loops, Functions, Arrays & Objects, ES6+ Features.

HTML5, CSS3, Flexbox & Grid, Responsive Design, Bootstrap/Tailwind Overview, DOM Manipulation, Events.

Introduction to Angular, CLI Setup, Project Structure, TypeScript Fundamentals, Components, Data Binding, Directives, Pipes, Event Binding, Two-Way Binding. Services & DI, Routing, Reactive Forms, Template-Driven Forms, HTTP Client & API Calls, Guards, Interceptors, State Management Overview, Build & Deploy.

Introduction to Node.js, Architecture, NPM, Node Modules, File System Module, Event Loop Concept, Building Basic Server.

Introduction to Express, Routing, Middleware, REST API Design, Error Handling, JWT Authentication, Role-Based Authorization.

Introduction to NoSQL, MongoDB Installation, CRUD Operations, Schema Design, Mongoose ODM, Data Modeling, Indexing.

Connect Angular with Node/Express, CORS, End-to-End CRUD Application, Form Validation, Authentication Flow (Login/Signup).

Unit Testing Basics, API Testing with Postman, Debugging Node Applications, Logging Best Practices.

Git & GitHub, Docker Basics, CI/CD Overview, Jenkins Overview, Deploy MEAN App (AWS/VPS), Nginx Basics.

Full Stack MEAN Project (Enterprise App/CRM), Architecture Best Practices, Performance Optimization, Portfolio Building, Resume Building, Mock Interviews, MEAN Interview Questions.
