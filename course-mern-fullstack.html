Course Overview

Full stack JavaScript with React (frontend), Node.js, Express (backend), and MongoDB. Build complete web applications with authentication, REST APIs, and deployment. Includes testing, DevOps overview, and a full MERN project (E-Commerce/CRM).

Course Curriculum

Web & Programming Fundamentals (1–10)

Introduction to Web Development, Client vs Server, How the Internet Works, JavaScript: Variables, Data Types, Operators, Control Statements, Loops, Functions, Arrays & Objects.

Frontend Basics (11–18)

HTML5, CSS3, Flexbox & Grid, Responsive Design, Bootstrap/Tailwind Overview, DOM Manipulation, Events, ES6+ Features.

React – Frontend (19–35)

Introduction to React, Project Setup, JSX, Functional Components, Props, State, Event Handling, Conditional Rendering, Lists & Keys. React Hooks (useState, useEffect), Custom Hooks, React Router, Form Handling, API Calls (Fetch/Axios), Context API, Performance Optimization, Build & Deploy.

Node.js Backend (36–42)

Introduction to Node.js, Architecture, NPM, Node Modules, File System Module, Event Loop Concept, Building Basic Server.

Express.js (43–49)

Introduction to Express, Routing, Middleware, REST API Design, Error Handling, JWT Authentication, Role-Based Authorization.

MongoDB (50–56)

Introduction to NoSQL, MongoDB Installation, CRUD Operations, Schema Design, Mongoose ODM, Data Modeling, Indexing.

Full Stack Integration (57–61)

Connect React with Node/Express, CORS, End-to-End CRUD Application, Form Validation, Authentication Flow (Login/Signup).

Testing & Quality (69–72)

Unit Testing Basics, API Testing with Postman, Debugging Node Applications, Logging Best Practices.

DevOps & Deployment (73–78)

Git & GitHub, Docker Basics, CI/CD Overview, Jenkins Overview, Deploy MERN App (AWS/VPS), Nginx Basics.

Real-Time Project (79–85)

Full Stack MERN Project (E-Commerce/CRM), Architecture Best Practices, Performance Optimization, Portfolio Building, Resume Building, Mock Interviews, MERN Interview Questions.
