Course Overview

Complete DevOps program: Linux, Git, Maven, Jenkins, Docker, Kubernetes, AWS, and monitoring. Includes AI add-on (AIOps, AI in CI/CD, log analysis). End-to-end CI/CD pipeline and real-time project. Resume building and mock interviews.

Course Curriculum

DevOps Fundamentals (1–7)

Introduction to DevOps, SDLC vs DevOps, DevOps Life Cycle, Principles & Benefits, Roles & Responsibilities, Waterfall vs Agile vs DevOps, Introduction to Linux for DevOps.

Linux for DevOps (8–16)

Installation & Basics, File System, Basic Commands, File Permissions, User & Group Management, Package Management, Process Management, Shell Scripting Basics, Cron Jobs.

Version Control (17–24)

Introduction to Git, Installation & Setup, Git Workflow, Commands (Clone, Add, Commit, Push, Pull), Branching & Merging, Merge Conflicts, GitHub, Pull Requests & Code Review.

Build & Dependency – Maven (25–29)

Introduction to Maven, Project Structure, Build Lifecycle, Dependency Management, Building Artifacts (JAR/WAR).

CI – Jenkins (30–38)

Introduction to Jenkins, Installation & Setup, Dashboard, Freestyle Project, Pipeline Jobs, Jenkinsfile (Pipeline as Code), Git Integration, Automated Build & Test, Email Notifications.

Containerization – Docker (39–48)

Introduction to Docker, Architecture, Installing Docker, Images, Containers, Dockerfile, Volumes, Networking, Docker Compose, Containerizing Applications.

Container Orchestration – Kubernetes (49–58)

Introduction to Kubernetes, Architecture, Installation (Minikube/K8s), Pods, Deployments, Services, ConfigMap & Secrets, Auto Scaling, Rolling Updates & Rollback, Helm Basics (optional).

Cloud – AWS (59–67)

Introduction to Cloud Computing, AWS Global Infrastructure, EC2, EBS & Storage, S3, IAM, VPC Basics, Load Balancer Basics, Deploy Application on AWS.

Monitoring & Logging (68–72)

Introduction to Monitoring, Prometheus Basics, Grafana Dashboards, ELK Stack Overview, Log Management Basics.

CI/CD End-to-End (73–77)

Build Complete CI/CD Pipeline, Git → Maven → Jenkins → Docker → K8s → AWS Flow, Automated Deployment, Blue-Green Deployment, Rolling Deployment Strategy.

AI Add-on for DevOps (78–83)

Introduction to AIOps, AI in CI/CD Pipelines, Smart Failure Prediction, AI-Based Log Analysis, Using ChatGPT for Shell Scripting, AI-Assisted YAML Generation.

Real-Time Project & Career (84–89)

End-to-End DevOps Project, Production-Like Pipeline Setup, Best Practices, Resume Building, Mock Interviews, DevOps Interview Questions.
