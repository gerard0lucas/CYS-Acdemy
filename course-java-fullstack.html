Course Overview

High-demand Java Full Stack program covering Core Java, React frontend, and Spring Boot backend. Build end-to-end applications with REST APIs, JPA, and deployment. Ideal for freshers and career switchers targeting Bangalore IT roles.

Course Curriculum

Introduction to Programming & Java, Installation & Environment Setup, Program Structure, Variables & Data Types, Operators, Control Statements, Loops, Arrays, Methods.

Introduction to OOPs, Class and Object, Constructors, Method Overloading, Inheritance, Method Overriding, Polymorphism, Encapsulation, Abstraction, Interfaces, Packages.

Exception Handling, String Handling, Java Collections Framework, Generics, Multithreading, Synchronization, File Handling, Java 8 Features (Lambda, Stream API), Optional Class, Date & Time API.

Introduction to Database, SQL Basics, Constraints, Joins, Subqueries, Indexes, Normalization, JDBC Introduction, JDBC CRUD Operations.

Web Fundamentals: HTML5, CSS3, Flexbox & Grid, Bootstrap, JavaScript Basics, DOM Manipulation, Events, ES6 Features. React Core: Introduction, Project Setup, JSX, Functional Components, Props, State, Event Handling, Conditional Rendering, Lists & Keys. React Advanced: Hooks (useState, useEffect), Custom Hooks, React Router, Form Handling, API Calls (Axios/Fetch), Context API, Performance Optimization, Build & Deploy.

Introduction to Spring, IoC Container, Dependency Injection, Spring Beans, Spring MVC, Spring Boot Setup, RESTful Web Services, Spring Data JPA, Hibernate Basics, Entity Mapping, CRUD with Spring Boot, Global Exception Handling.

Spring Security Basics, JWT Authentication, Role-Based Authorization.

Connect React with Spring Boot, CORS Configuration, End-to-End CRUD Application, Form Validation (Frontend + Backend).
